## Maintainer Responsibilities

- Review and merge pull requests
- Triage issues and security reports
- Release management
- Documentation updates
- Community engagement

## Becoming a Maintainer

We welcome new maintainers! To become a maintainer:

1. Contribute significantly to the project (code, docs, reviews)
2. Demonstrate understanding of the security model
3. Show consistent engagement over time
4. Be nominated by an existing maintainer
