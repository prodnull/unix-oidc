**Part A: Create test/tests/test_break_glass_e2e.sh**

Create an executable bash script. Follow the pass()/fail() pattern from test_break_glass_fallback.sh.

Variables: `COMPOSE_FILE="${COMPOSE_FILE:-docker-compose.test.yaml}"`, `KEYCLOAK_URL`, `TEST_HOST` (SSH target), `BREAK_GLASS_USER` (default: `breakglass`), `NORMAL_USER_IN_GROUP` (default: `testuser`), `NORMAL_USER_NOT_IN_GROUP` (default: `testuser2` or create a user not in login_groups).

Script structure:

1. **Check compose stack running** — skip if not.

2. **Test 1 — Break-glass when IdP is down:**
   - Stop Keycloak: `docker compose -f "$COMPOSE_FILE" stop keycloak`.
   - Wait 2 seconds for health checks to settle.
   - Attempt SSH as break-glass user with local password via `sshpass` or expect: `sshpass -p "$BREAK_GLASS_PASS" ssh -o StrictHostKeyChecking=no testhost "echo BREAK_GLASS_OK"`.
   - Assert: output contains "BREAK_GLASS_OK" and exit code 0.
   - Assert: `docker compose exec test-host grep "BREAK_GLASS_AUTH\|break_glass" /var/log/auth.log` returns a line.
   - Restart Keycloak: `docker compose -f "$COMPOSE_FILE" start keycloak` and wait for healthy.
   - pass() on success.

3. **Test 2 — NSS group policy denial:**
   - Attempt SSH as `testuser2` (a user configured in SSSD/NSS but NOT in `login_groups`).
   - SSH should fail: `ssh -o BatchMode=yes -o StrictHostKeyChecking=no testhost "echo SHOULD_NOT_REACH"` exits non-zero.
   - Assert: exit code non-zero.
   - Assert: `docker compose exec test-host grep -E "group policy|not in login_groups|access denied" /var/log/auth.log` returns a line.
   - pass() on assertion (SSH rejected = correct behavior).

4. **Test 3 — Normal user IN login_group succeeds:**
   - Acquire valid OIDC token for `testuser` (who IS in login_groups).
   - Run SSH_ASKPASS flow: `SSH_ASKPASS=unix-oidc-agent SSH_ASKPASS_REQUIRE=force ssh ... testuser@testhost "echo AUTH_OK"`.
   - Assert: output contains "AUTH_OK".
   - pass() on success.

5. Print summary.

If `sshpass` is unavailable, use a here-document or expect script.
