**TpmSigner struct:**
```rust
pub struct TpmSigner {
    persistent_handle: u32,
    tcti_conf: String,        // "tabrmd" or "device" or "mssim"
    pin_cache: PinCache,
    thumbprint: String,
    public_key_jwk: serde_json::Value,
}
```

**Static function `TpmSigner::probe_p256(tcti: &str) -> anyhow::Result<()>`** (HW-05):
1. Build `TctiNameConf` from string: "tabrmd" -> `TctiNameConf::Tabrmd(TabrmdConfig::default())`, "device" -> `TctiNameConf::Device(DeviceConfig::default())`, "mssim" -> `TctiNameConf::Mssim(MssimConfig::default())`.
2. Create `Context::new(tcti_conf)?`
3. Call `ctx.get_capability(CapabilityType::EccCurves, 0, 32)?`
4. Check if `EccCurve::NistP256` is in the returned list.
5. If not: `anyhow::bail!("This TPM does not support P-256 (NistP256 ECC curve). DPoP requires ES256 (P-256 ECDSA). Cloud vTPMs may not support all curves.")`.

**Static function `TpmSigner::provision(config: &SignerConfig) -> anyhow::Result<(u32, serde_json::Value, String)>`:**
1. Get TpmConfig from config (or defaults: tcti="tabrmd", handle=0x81000001).
2. Call `probe_p256(tcti)?` first.
3. Create Context, create primary unrestricted signing key under Owner hierarchy:
   - Use `tss_esapi::utils::create_unrestricted_signing_ecc_public` if available, else build `PublicEccParametersBuilder` manually with `EccScheme::EcDsa { hash_scheme: HashScheme::new(HashingAlgorithm::Sha256) }` and `EccCurve::NistP256`.
   - `ctx.create_primary(Hierarchy::Owner, ecc_public, None, None, None, None)?`
4. Make persistent: `ctx.evict_control(Provision::Owner, key_handle, Persistent::Persistent(PersistentTpmHandle::new(handle)?))?`
5. Read public key from the created key's public area (it's returned by create_primary). Extract x,y from the ECC parameters.
6. Build JWK and compute thumbprint from x,y (same pattern as YubiKeySigner: base64url encode coords, build canonical JSON, SHA-256).
7. Return (handle, jwk, thumbprint).

**Constructor `TpmSigner::load(config: &SignerConfig) -> anyhow::Result<Self>`:**
1. Get TpmConfig from config (defaults as above).
2. Create Context, load key from persistent handle via `ctx.tr_from_tpm_public(TpmHandle::Persistent(PersistentTpmHandle::new(handle)?))`.
3. Read public key from TPM: `ctx.read_public(key_handle)?` returns the Public structure with ECC parameters containing x,y.
4. Build JWK and thumbprint from x,y.
5. Build PinCache with config timeout.
6. Return TpmSigner.
7. If TPM not available: "TPM not available. Check that tpm2-abrmd is running (`systemctl status tpm2-abrmd`) or specify a different TCTI in signer.yaml."

**DPoPSigner implementation:**
- `thumbprint()` -> stored thumbprint
- `public_key_jwk()` -> stored JWK
- `sign_proof()`:
  1. `build_dpop_message(&self.public_key_jwk, method, target, nonce)?`
  2. Create new Context per operation (no persistent context held):
     - Build TctiNameConf from self.tcti_conf
     - `Context::new(tcti)?`
  3. Load key handle from persistent handle.
  4. SHA-256 hash the message in agent: `sha2::Sha256::digest(message.as_bytes())`
  5. Build `Digest::try_from(hash_bytes.as_slice())?`
  6. Build signing scheme: `SignatureScheme::EcDsa { hash_scheme: HashScheme::new(HashingAlgorithm::Sha256) }`
  7. Build null HashcheckTicket for unrestricted signing key — try `HashcheckTicket::null()` first; if not available, construct from raw TPMT_TK_HASHCHECK with `tag: TPM2_ST_HASHCHECK, hierarchy: TPM2_RH_NULL, digest: Default::default()`.
  8. `ctx.sign(key_handle, digest, scheme, validation)?`
  9. Extract r||s from `Signature::EcDsa(ecc_sig)`:
     - `r = ecc_sig.signature_r().value()` and `s = ecc_sig.signature_s().value()`
     - Left-pad each to exactly 32 bytes (P-256). Use helper function `pad_to_32(bytes: &[u8]) -> [u8; 32]`.
     - Concatenate: `[r_padded || s_padded]` = 64 bytes.
  10. `assemble_dpop_proof(&message, &sig_bytes)?`

**Helper `pad_to_32(bytes: &[u8]) -> [u8; 32]`:** (public for testing)
```rust
pub fn pad_to_32(bytes: &[u8]) -> [u8; 32] {
    let mut out = [0u8; 32];
    let len = bytes.len().min(32);
    let start = 32 - len;
    out[start..].copy_from_slice(&bytes[bytes.len().saturating_sub(32)..]);
    out
}
```
