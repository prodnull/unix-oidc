<objective>
Add a background sweep task to the agent daemon that periodically removes expired and corrupt session records from /run/unix-oidc/sessions/.

Purpose: Session records from crashed sshd workers that never sent SessionClosed IPC will accumulate indefinitely. The sweep is the safety net that prevents /run/unix-oidc/sessions/ from growing without bound.
Output: New sweep.rs module, config extension, and spawn wiring in serve_with_listener.
</objective>
